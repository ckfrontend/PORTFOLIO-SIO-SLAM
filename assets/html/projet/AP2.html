Présentation


Cette application permet de gérer les documents (livres, DVD, revues) d'une médiathèque. Elle a été codée en C# sous Visual Studio 2019. C'est une application de bureau, prévue d'être installée sur plusieurs postes accédant à la même base de données. L'application exploite une API REST pour accéder à la BDD MySQL. Des explications sont données plus loin, ainsi que le lien de récupération.

Contexte



MediatekDocuments

Les médiathèques modernes gèrent une grande diversité de documents, allant des livres traditionnels aux supports multimédias comme les DVD et les revues. La gestion efficace de ces ressources nécessite une application robuste et intuitive, capable de centraliser les informations et de permettre un accès facile et rapide aux données.

Objectifs et Utilité pour les Utilisateurs :

L'objectif principal de cette application est de fournir un outil complet pour les gestionnaires de médiathèques, leur permettant de cataloguer, de prêter et de gérer les retours de documents.
