SQL Server

SQL est un langage informatique normalisé servant à exploiter des bases de données relationnelles. La partie langage de manipulation des données de SQL permet de rechercher, d'ajouter, de modifier ou de supprimer des données dans les bases de données relationnelles.



Microsoft SQL Server est un système de gestion de base de données (SGBD) en langage SQL incorporant entre autres un SGBDR (SGBD relationnel) développé et commercialisé par la société Microsoft. Il fonctionne sous les OS Windows et Linux (depuis mars 2016).

Points forts de SQL Server :

• SQL Server Management Studio pour gérer facilement les bases de données (création de tables, requêtes,…)
• Intégration avec d’autres produits Microsoft
• Très bonnes performances en général sous Windows
• Options de sécurité poussées
• Compression des données et des sauvegardes
